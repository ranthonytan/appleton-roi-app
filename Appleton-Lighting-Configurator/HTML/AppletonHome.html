The AppletonTM Lighting Retrofit Calculator helps users visualize savings accomplished by retrofitting their current lighting system to an AppletonTM LED solution.

1. Input existing costs associated with maintaining current legacy lighting installation (high intensity discharge (HID), fluorescent, incandescent). Costs include labor for maintenance personnel, electricity, replacement lamps and ballasts, and equipment rental to perform lighting maintenance. Select an AppletonTM LED solution.

2. Once the data is entered, compare existing costs to proposed AppletonTM LED solution and see the reduced environmental impact.

3. Output results can be printed or shared via e-mail.
